   + 서버 이전 미루는 문제, 매번 반복되는 서버 세팅 과정, 낭비되는 시간

   + AI빌드 에러 수정 지원
  4. 서비스 구조 및 사용 흐름 
   + 사용자 등록 정보와 해당 정보를 통한 서버 세팅과 자동 배포 구축 과정
