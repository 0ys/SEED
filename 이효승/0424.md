- 프로젝트 텍스트 스타일 생성 및 적용
- 대시보드 화면 UI 및 내용 수정

  ![alt text](image/image9.png)

  - 프로젝트 구분없이 캘린더에 해당 일자에 발생한 이벤트 표시 -> 클릭시 관련 프로젝트 상세 페이지로 이동

- 프로젝트 상세화면 표시 정보 논의
